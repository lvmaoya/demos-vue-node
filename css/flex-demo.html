📚 Flex 布局基础概念

Flexbox（弹性盒子）是一种一维布局方法，用于在容器中分配空间和对齐项目。

核心概念：

  • Flex Container（容器）：设置了 display: flex 的父元素
  • Flex Item（项目）：容器内的直接子元素
  • Main Axis（主轴）：项目排列的主要方向
  • Cross Axis（交叉轴）：垂直于主轴的方向

容器属性：

属性 说明 可选值
display 定义为flex容器 flex | inline-flex
flex-direction 主轴方向 row | row-reverse | column | column-reverse
justify-content 主轴对齐方式 flex-start | flex-end | center | space-between | space-around | space-evenly
align-items 交叉轴对齐方式 stretch | flex-start | flex-end | center | baseline
flex-wrap 是否换行 nowrap | wrap | wrap-reverse

项目属性：

属性 说明 默认值
flex-grow 放大比例 0
flex-shrink 缩小比例 1
flex-basis 基础大小 auto
flex 上述三个属性的简写 0 1 auto
align-self 单独的对齐方式 auto

🔟 实际应用 - 完美居中布局

使用 Flex 布局实现水平垂直完美居中。

.center-container { display: flex; justify-content: center; align-items: center; height: 200px; }

🎯 完美居中

无论容器大小如何变化，内容始终保持居中

📝 Flex 布局总结

🎯 何时使用 Flex 布局：

  • 需要在一个方向上排列元素
  • 需要灵活的空间分配
  • 需要简单的对齐控制
  • 创建响应式布局
  • 实现居中布局

💡 最佳实践：

  • 优先使用 flex 简写属性而不是单独设置 flex-grow、flex-shrink、flex-basis
  • 使用 gap 属性设置项目间距（现代浏览器支持）
  • 结合媒体查询实现响应式设计
  • 对于二维布局考虑使用 CSS Grid
  • 注意浏览器兼容性，特别是旧版本IE

🔧 常用组合：

  • display: flex; justify-content: center; align-items: center; - 完美居中
  • display: flex; justify-content: space-between; - 两端对齐
  • display: flex; flex-wrap: wrap; gap: 20px; - 响应式网格
  • flex: 1; - 填充剩余空间
